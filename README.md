# 🍔 YummyGo - Premium Food Delivery Experience

[![Flutter](https://img.shields.io/badge/Flutter-Framework-blue)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-Language-blue)](#)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**YummyGo** is a comprehensive food delivery application, designed with the latest technologies to provide a luxurious and seamless user experience (UI/UX). The app connects food lovers with fine dining restaurants, offering a unique tasting journey that starts with browsing and discovery, all the way to order tracking and delivery.

---

## ✨ Key Features

- **Stunning Visual Experience**: A modern, elegant, and carefully crafted UI/UX designed for maximum visual comfort.
- **Dark/Light Mode Support**: Seamless and dynamic switching between dark and light themes.
- **Professional Browsing**: Detailed and beautiful displays for restaurants, food menus, and ingredients.
- **Smart Cart & Secure Checkout**: Smooth and intuitive steps to complete the purchase process.
- **Live Order Tracking**: An interactive interface to track the order status step-by-step.
- **Advanced State Management & Routing**: Built with `Provider` for robust state management and `GoRouter` to ensure professional screen transitions.

## 🛠️ Tech Stack

This project is built using modern technologies and libraries in the app development ecosystem:
- [Flutter](https://flutter.dev/) - UI Framework.
- [Provider](https://pub.dev/packages/provider) - State Management.
- [GoRouter](https://pub.dev/packages/go_router) - Declarative and complex routing.
- [Google Fonts](https://pub.dev/packages/google_fonts) - Professional typography styling.
- [Flutter Animate](https://pub.dev/packages/flutter_animate) - Smooth, premium animations and transitions.
